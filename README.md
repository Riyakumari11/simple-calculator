# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean user interface.

---

## 🚀 Features

* Perform basic calculations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Clear button to reset input
* Responsive design (works on desktop & mobile)
* User-friendly interface

---

## 🛠️ Technologies Used

* **HTML** – Structure of the calculator
* **CSS** – Styling and layout
* **JavaScript** – Logic and functionality

---

## 📂 Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
```

---

## 💡 How It Works

* User clicks buttons to input numbers and operators
* JavaScript handles the calculation logic
* Result is displayed on the screen dynamically

---

## ▶️ How to Run

1. Download or clone this repository

   ```
   git clone https://github.com/your-username/calculator.git
   ```
2. Open `index.html` in your browser
